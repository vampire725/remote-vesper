# 使用方法: 复制内容到 .env 文件并修改密码
# ===========================================

# ==========================================
# Elasticsearch 用户密码配置
# ==========================================

# elastic 超级用户密码（必须设置）
# 建议使用强密码，包含大小写字母、数字和特殊字符
ELASTIC_PASSWORD=ElasticSearch2024!

# kibana_system 用户密码（用于 Kibana 连接 Elasticsearch）
# 如果不使用 Kibana，可以设置为任意值
KIBANA_PASSWORD=Kibana2024!

# ==========================================
# 使用说明
# ==========================================
# 1. 创建 .env 文件: 将上述内容复制到 .env 文件中
# 2. 修改密码: 将默认密码改为您自己的强密码
# 3. 密码要求:
#    - 长度至少 8 位
#    - 包含大小写字母、数字和特殊字符
#    - 不要使用常见密码或个人信息
# 4. 安全提示:
#    - 定期更换密码
#    - 确保 .env 文件不被提交到版本控制系统
#    - 限制 .env 文件的访问权限
